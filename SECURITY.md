### English

If you discover a security vulnerability in this project, please report it responsibly:

1. **DO NOT** create a public GitHub issue for security vulnerabilities
2. Email the maintainer directly or use [GitHub Security Advisories](https://github.com/DanOps-1/Xray-VPN-OneClick/security/advisories/new)
3. Include as much detail as possible:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Suggested fix (if any)

We will respond within **48 hours** and work with you to understand and resolve the issue.
